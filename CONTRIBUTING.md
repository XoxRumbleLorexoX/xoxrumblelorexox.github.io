# Contributing

## Standards
- Keep JavaScript in `js/`, styles in `css/`, and data in `data/`.
- Prefer compressed assets and keep individual images below ~500KB when possible.
- Run `npm run lint` and `npm run test:smoke` before opening PRs.

## PR/Commit guidance
- Use focused commits and include a summary of visible user impact.
- Update `CHANGELOG.md` and `TODO.md` when behavior changes.
