# X0xRumbleLorex0X Portfolio

Interactive portfolio site with single-page pane navigation, project showcases, PDF decks, and dynamic overlays.

## Local development

```bash
python -m http.server 4173
# open http://127.0.0.1:4173/index.html
```

## Quality commands

```bash
npm ci
npm run lint
npm run format:check
npm run build
npm run test:smoke
```

## Deployment

- GitHub Pages can serve directly from the repository root.
- For optimized assets, deploy from `dist/` generated by `npm run build`.

## Architecture overview

- `index.html`: page shell and pane structure.
- `css/`: visual styles and motion/accessibility rules.
- `js/`: pane logic, overlays, model viewer, gallery, and PDF renderer.
- `data/`: cached content payloads for thoughts and Medium posts.
- `tools/`: utility scripts (build + media conversion helpers).
