# Data schemas

## `data/thoughts.json`
Array of objects:
- `title` (string, required)
- `body` (string, required)
- `date` (ISO date string, optional)

## `data/medium.json`
Array of objects:
- `title` (string, required)
- `link` (URL string, required)
- `description` (string, optional)
- `pubDate` (date string, optional)
